Nossa Missão

Aprendizagem baseada em projetos

O SENAI, em parceria com a JPMorgan Chase Foundation, quer formar jovens profissionais em tecnologia da informação por meio de metodologias inovadoras e apoiar sua entrada no mercado de trabalho.

O curso é gratuito e, além de aprender o conteúdo que realmente é utilizado no dia a dia das empresas, os alunos terão contato com profissionais do mercado, a possibilidade de desenvolver projetos e de trabalhar com mentores.

[ Nosso formato ]

O Code Experience é composto por 4 cursos que serão desenvolvidos com uma abordagem transdisciplinar, refletindo a estrutura de uma equipe de desenvolvimento de software.

[ Estrutura dos cursos ]

Cada curso será composto por três módulos:

Base: Conceitos essenciais a todo desenvolvedor de software.

Específico: Formação direcionada para as diferentes tecnologias utilizadas no mercado de trabalho.

Transdisciplinar: Os alunos formarão equipes com membros dos 4 cursos para desenvolverem um projeto real de software.

[ Sobre o Hackaton ]

Durante o curso, você vai participar junto com sua equipe de um Hackaton, uma maratona de programação para criar soluções inovadoras para o mercado de TI.

Na reta final, você vai desenvolver os projetos no Hackaton e colocar em prática tudo o que aprendeu, tendo a oportunidade de mostrar o seu trabalho para grandes empresas.
